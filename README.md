# AIRI-Intern

아주대학교 현장실습으로 인공지능연구원 AIRI 빅데이터개발팀에서 인턴으로 지원하기 위해 만들었다.  



계획서에 보면 금융/시장 데이터 수집/처리 및 시각화를 위한 데이터처리/백엔드/프론트엔드 개발이라 적혀있다. 
합격에 중요한 요소로 당장 구현이 가능한가라고 생각한다. 그러기 위해 빠르게 서비스를 개발해보고 이를 증명해보고자 한다.
